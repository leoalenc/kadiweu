# Texto Livre abstract evaluation snapshot

This directory preserves the CoNLL-U drafts, gold treebank, and comparison reports
used for the LAS/UAS and related metrics reported in the abstract submitted to the
Texto Livre special issue on IA and Linguistics.

The current repository may contain newer versions of the converter, override
resources, source JSON files, draft treebanks, and gold treebank. The files in this
directory are frozen for reproducibility.
