# Notes

This snapshot corresponds to the evaluation reported in the abstract submitted
to the Texto Livre special issue "IA and Linguistics".

The snapshot contains:

- frozen gold treebank;
- frozen converter outputs;
- comparison reports;
- token mismatch reports.

Subsequent commits may modify:

- the converter;
- override resources;
- source JSON files;
- the gold treebank.

The files contained in this release directory must therefore be considered the
authoritative reference for the reported LAS, UAS, and related metrics.
